# Safe Yatri

## Smart Tourist Safety Monitoring & Incident Response System

Safe Yatri is an AI-powered tourist safety monitoring platform designed to improve tourist security through real-time monitoring, geo-fencing, incident management, and emergency response coordination.

## Features

- Tourist Registration & Management
- Safety Score Monitoring
- Geo-Fencing & Restricted Zone Alerts
- Live Tourist Tracking
- Incident Reporting & Management
- Emergency Alert System
- eFIR Generation
- Dashboard Analytics
- Role-Based Access Control

## Technology Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- React Router
- Authentication System
- Dashboard Analytics

## Modules

### Tourist Management
Manage tourist registration, profiles, and travel information.

### Geo-Zone Monitoring
Monitor restricted and high-risk areas through geo-fencing.

### Incident Response
Track incidents and coordinate emergency response actions.

### Live Tracking
Monitor tourist activity and movement through interactive maps.

### Alert System
Generate and manage real-time alerts.

### eFIR Management
Support digital reporting and documentation of incidents.
